# File I/O

This project covers low-level file input/output operations in C using POSIX system calls at Holberton School.

## Concepts

- File descriptors and their role in POSIX systems
- Reading and writing files using `open`, `read`, `write`, and `close`
- Memory management when handling file buffers
- Error handling for file operations

## Files

- `main.h` — header file with function prototypes
- `0-read_textfile.c` — reads a text file and prints it to the standard output
