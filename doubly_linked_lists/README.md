# Doubly Linked Lists

This project covers doubly linked lists in C at Holberton School.

## Data structure

Each node in a dlistint_t contains:
- An integer value `n`
- A pointer `prev` to the previous node
- A pointer `next` to the next node

## Files

- `lists.h` — header file with the dlistint_t structure and function prototypes
- `0-print_dlistint.c` — prints all the elements of a dlistint_t list
